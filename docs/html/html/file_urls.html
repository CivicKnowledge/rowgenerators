File Urls¶

File URLs represent resources that are acessible on the local file system.

Because these URLs are assumed to be local, the resource already exists on the local file system, and the FileUrl.get_resource() just returns self

class rowgenerators.appurl.file.FileUrl(url=None, downloader=None, **kwargs)[source]¶

FileUrl is the baseclass for URLs that reference a general file, assumed to be local to the file system.

This documentation only describes the differences in implementation from the super class. See the documentation for the superclass, appurl.Url for the default implementations.
